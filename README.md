# React E-commerce UI

A responsive e-commerce frontend built with React and Vite.

## Features

- Home Page
- Product Listing
- Product Details
- Shopping Cart
- Login Page
- Search Products
- Category Filters
- Add to Cart (Context API)
- Responsive Design

## Tech Stack

- React
- React Router DOM
- Context API
- CSS3
- DummyJSON API
- React Icons

## Installation

```bash
npm install
npm run dev
```
